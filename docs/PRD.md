### PRD：Photo Watermark 2（Windows 本地应用，Python）

#### 一、产品概述
- **产品定位**: 一款面向 Windows 用户的本地图片批量加水印工具，支持文本与图片水印、实时预览、模板保存与管理，一键导入导出，满足摄影师、设计师、内容创作者等对版权保护与品牌露出需求。
- **目标用户**: 需要高效批量处理图片水印的个人或团队用户。
- **核心价值**: 易用、高效、可靠的本地处理，无需联网，保证隐私与稳定性能。

#### 二、范围与目标
- **必做目标**
  - 单张/批量图片导入（含文件夹）。
  - 支持 JPEG、PNG（含透明通道）、建议支持 BMP、TIFF。
  - 文本水印（内容、透明度、位置、九宫格预设、拖拽移动、实时预览）。
  - 导出为 JPEG/PNG；输出目录不可为原目录。
  - 导出命名规则（保留原名/自定义前缀/自定义后缀）。
  - 水印模板保存、加载、删除；启动自动加载上次设置或默认模板。
- **可选高级功能（若时间允许）**
  - 文本水印：字体选择、字号、粗体、斜体、颜色选择器、阴影/描边。
  - 图片水印：透明 PNG、比例缩放、透明度调节。
  - 旋转：任意角度旋转。
  - 导出 JPEG 质量滑块、导出尺寸缩放（宽/高/百分比）。
- **非目标（本期不做）**
  - 云同步、多人协作、在线字体下载、滤镜/美化、EXIF 编辑。

#### 三、用户画像与关键场景
- **用户画像**
  - 摄影师：批量为样片添加半透明水印，防止盗图。
  - 自媒体/电商运营：为商品图添加品牌 Logo 与文本说明。
  - 设计师：为设计稿导出稿添加标记。
- **关键用户场景**
  1) 拖拽一批图片 → 输入文本水印 → 选右下角位置 → 调整透明度 → 选择输出目录与命名规则 → 批量导出。
  2) 选择图片 Logo 作为水印 → 设定缩放与透明度 → 保存为模板 → 下次打开自动应用 → 一键导出。
  3) 切换图片列表查看不同图的预览 → 微调位置与样式 → 导出为 PNG 保持透明背景区域。

#### 四、功能需求说明
- **1. 文件处理**
  - 1.1 导入图片
    - 支持拖拽单图/多图/文件夹。
    - 支持文件选择器多选及选择文件夹。
    - UI 左侧列表显示缩略图+文件名；可点击切换预览目标。
    - 重复导入自动去重（基于绝对路径）。
  - 1.2 支持格式
    - 输入：JPEG、PNG（含透明）、强烈建议 BMP、TIFF。
    - 输出：JPEG 或 PNG，用户可选。
  - 1.3 导出图片
    - 必须选择非原目录作为输出目录（若相同需阻止并提示）。
    - 命名规则：
      - 保留原文件名。
      - 自定义前缀（如“wm_”）。
      - 自定义后缀（如“_watermarked”）。
    - 可选：JPEG 质量 0-100；尺寸缩放（按宽/高/百分比，保持/可选保持长宽比）。
- **2. 水印类型**
  - 2.1 文本水印（必做）
    - 文本内容可编辑。
    - 透明度 0-100%。
    - 位置：九宫格预设（四角、边中、正中）；支持拖拽调整任意位置。
    - 可选：字体（系统字体列表）、字号、粗体/斜体、颜色选择器、阴影/描边。
  - 2.2 图片水印（可选）
    - 从本地选择一张图片（建议 PNG、支持透明）。
    - 支持按比例缩放或自由缩放。
    - 透明度 0-100%。
- **3. 布局与交互**
  - 实时预览：参数变动即刻在主预览生效；切换列表项时预览对应图片。
  - 位置控制：九宫格快捷定位+鼠标拖拽；显示位置辅助线或坐标（可选）。
  - 旋转：滑块/输入框设定角度（可选）。
- **4. 配置管理**
  - 模板：保存当前所有水印参数为模板（文本/图片水印、字体样式、颜色、位置、大小、透明度、旋转、导出设置等）。
  - 模板管理：列表展示、加载、删除、重命名。
  - 启动加载：自动加载上次退出前的设置或默认模板（可在设置中配置）。

#### 五、非功能需求
- **性能**: 
  - 100 张 4K 图片批量导出在常见 Windows 笔记本上应在合理时间完成（目标：≤3 分钟，视水印复杂度与压缩率）。
  - 预览缩略图和主预览采用缓存，避免重复解码；导出处理使用原图。
- **兼容性**: Windows 10/11；支持高 DPI 缩放。
- **稳定性**: 处理失败有详细错误提示，不崩溃；导出过程可取消。
- **隐私**: 全程本地处理，不上传任何数据。
- **可用性**: 简洁 UI，重要选项默认合理值；支持快捷键（可选）。

#### 六、信息架构与主要界面
- **主窗口布局**
  - 左侧：图片列表（缩略图 + 文件名，支持多选、右键移除、清空）。
  - 中央：主预览（自适应缩放，显示当前水印效果，支持拖拽水印）。
  - 右侧：属性面板
    - Tab1 文本水印：开关、文本输入、透明度、字体/字号/样式/颜色（可选）、阴影/描边（可选）、旋转（可选）。
    - Tab2 图片水印（可选）：开关、选择图片、缩放、透明度、旋转（可选）。
    - Tab3 导出设置：输出目录、命名规则、格式选择、JPEG 质量（可选）、尺寸缩放（可选）。
    - Tab4 模板：保存、加载、删除、自动加载开关。
  - 顶部工具栏：导入、选择文件夹、全选/全不选、导出、设置。
  - 底部状态栏：当前选中/总数、处理进度、错误提示。

#### 七、技术方案（建议）
- **语言/框架**: Python 3.10+；GUI 使用 PySide6（或 PyQt5）。
- **图像处理**: Pillow（PIL）用于加载、绘制文本、水印合成、缩放、旋转、导出；必要时用 `ImageQt` 与 Qt 预览联动。
- **字体**: 使用系统字体（Windows 字体目录扫描）或 Qt 字体对话框。
- **实时预览**: 基于原图生成适配窗口大小的预览位图；水印参数改变时仅重绘叠加层；使用内存缓存减少 IO。
- **拖拽**: Qt 鼠标事件计算水印矩形，支持吸附九宫格（可选）。
- **模板存储**: JSON 文件，默认存放 `%APPDATA%/PhotoWatermark2/templates/`；最近一次配置保存在 `last.json`。
- **配置校验**: 导出前校验输出目录不等于任一输入目录；命名冲突则自动加序号后缀或提示覆盖策略。
- **打包发布**: PyInstaller 生成单文件/单目录 EXE；附带依赖与运行库说明；提供 GitHub Release 可直接下载运行。
- **日志**: `logging` 到本地文件（便于问题排查，用户可关闭）。

#### 八、数据模型（示意）
- `ProjectState`
  - `images: List[ImageItem]`（原路径、缩略图缓存、元数据）
  - `watermark_text: TextWatermarkConfig`
  - `watermark_image: ImageWatermarkConfig`
  - `export_options: ExportOptions`
  - `active_template: str | None`
- `TextWatermarkConfig`
  - `enabled: bool, content: str, opacity: int(0-100)`
  - 可选：`font_family, font_size, bold, italic, color, shadow, stroke, rotation`
  - `position_mode: 'grid'|'free', grid_slot: 0..8, offset_x, offset_y`
- `ImageWatermarkConfig`（可选）
  - `enabled: bool, path: str, scale: float, opacity: int, rotation: float`
  - `position_mode, grid_slot, offset_x, offset_y`
- `ExportOptions`
  - `output_dir: str, format: 'jpeg'|'png'`
  - `naming: {'mode':'keep'|'prefix'|'suffix','value':str}`
  - 可选：`jpeg_quality: int, resize_mode: 'none'|'width'|'height'|'percent', resize_value: int`

#### 九、关键交互与状态流
- 导入 → 列表去重 → 生成缩略图 → 选中图片 → 右侧修改水印参数 → 中央预览实时刷新。
- 切换图片时沿用统一水印配置（默认对所有图片生效）；后期可选“每图单独微调”（非本期）。
- 导出时遍历图片，基于原图生成目标，命名规则生成文件名，写入输出目录。

#### 十、错误与边界处理
- 不支持/损坏图片：跳过并在状态栏/弹窗提示，日志记录。
- 输出目录为空/原目录/无权限：阻止并提示。
- 字体缺失：回退到默认字体（如 Segoe UI）并提示。
- 图片水印文件不存在：禁用图片水印并提示。
- 大图内存占用：采用预览缩放与处理分步、回收缓存。

#### 十一、可测试性与验收标准
- 单测：命名规则、路径校验、模板读写、图像叠加核心算法。
- 手测用例：
  - 导入多格式（JPEG/PNG/BMP/TIFF）成功显示缩略图。
  - 文本水印透明度与位置变更预览实时更新。
  - 九宫格一键定位准确；拖拽后坐标生效。
  - 导出为 JPEG/PNG，命名规则正确，JPEG 质量生效（可选）。
  - 输出目录非原目录校验有效。
  - 模板保存/加载/删除与启动自动加载正确。
- 验收：上述用例全部通过；100 张 4K 导出时间符合目标（环境说明）。

#### 十二、项目管理与里程碑（建议）

- M2（增强，可选）：
  - 字体与颜色、阴影/描边、图片水印、旋转、JPEG 质量、尺寸缩放。

#### 十三、发布与分发
- 构建：PyInstaller（`--noconsole` 生产版），附带 `requirements.txt`。
- 资产：`README.md` 使用说明、快捷键、常见问题；示例图片。
- GitHub Actions（可选）：自动构建 Windows artifact 与发布 Release。

#### 十四、风险与应对
- 大图/批量性能：采用缩略图预览与批处理、进度反馈、可取消。
- 字体兼容与中文渲染：优先系统字体，渲染前测试回退策略。
- 打包体积：精简依赖；必要时单目录分发替代单文件。
- 路径与权限（含中文路径）：全流程使用 Unicode 路径与异常捕获。
